# ForP 뽀삐

## 💡 프로젝트 개요: 반려동물 보호자를 위한 정보 제공 및 커뮤니티 서비스

반려동물 보호자를 위한 정보 제공 및 커뮤니트 중심의 웹 플랫폼을 구축한다.
이벤트 정보, 동물병원 지도, 게시판, 마이페이지 및 반려 동물 상태 계산 기능을 제공하여 사용자 편의성을 높인다.
